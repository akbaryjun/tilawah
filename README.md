# Tilawah V1

Tilawah is a complete Al-Quran website with transliteration and Indonesian translation.

## Getting Started

### Prequisites

Node JS

### Installing

```
npm install
```

### Run program locally

* How to run the program
```
npm run dev
```

## Features
- [x] Bookmark ayah
- [x] Audio included
- [x] Settings
  * Theme
  * Show/Hide Transliteration
  * Audio Auto Scroll
- [x] PWA
